HTML structure

HTML is used to create a structure of a webpage.The DOCTYPE html declaration tells the browser that the document is written in HTML5. The html tag is the root element of the page. The head tag contains information about the webpage such as the title which shows on the browser tab. The meta charset="UTF-8" tag helps display characters correctly, while the viewport meta tag makes the webpage responsive on different devices. The title tag sets the name shown on the browser tab. The body tag contains all the visible content of the webpage.
